# آموزش حلقه ها و بولین


- for loop
```C++
for ( /* initialization */ ; /* condition */ ; /* iteration */ )
{
   /* body */
}
```

- while loop
```C++
while ( /* condition */ )
{
   /* body */
}
```

- do while loop
```C++
do
{
   /* body */
}
while ( /* condition */ );
```

- loop control
```C++
continue; // Skip the loop
break; // break the loop
```

- Boolean
```C++
bool name = true;
bool name = false;
```
