# آشنایی با متغیر و انواع داده

int  = مقادیر عددی  
```C++
int esm = 1;
esm = 2;
```
float = مقادیر اعشاری 
```C++
float esm = 10.5;
esm = 12.1;
```
char و string = مقادیر متنی
```C++
char esm[12] = "text";
esm = "some text";
```
